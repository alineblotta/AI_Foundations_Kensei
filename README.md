# AI_Foundations_Kensei

## Descrição

O projeto **AI_Foundations_Kensei** tem como objetivo capacitar profissionais em uma abordagem AI-First para Cibersegurança, promovendo o desenvolvimento de competências técnicas e a criação de um portfólio atrativo. O projeto explora o uso de Inteligência Artificial, Python e Pandas para análise de dados, além de automação de processos com n8n, integração de APIs de LLMs e desenvolvimento de aplicações interativas com Streamlit.

## Tecnologias Utilizadas

- **Python 3.10+**
- **Pandas**
- **n8n**
- **APIs de LLMs (Large Language Models)**
- **Streamlit**
- **Jupyter Notebook** (opcional para experimentação)
- **Docker** (opcional para orquestração e deploy)

## Instalação e Execução

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/AI_Foundations_Kensei.git
   cd AI_Foundations_Kensei
   ```

2. **Crie e ative um ambiente virtual:**
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```

3. **Instale as dependências:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Execute os notebooks ou scripts Python conforme instruções nos diretórios correspondentes.**

5. **Para automação com n8n:**
   - Instale e configure o n8n conforme a [documentação oficial](https://docs.n8n.io/).
   - Importe os workflows disponíveis na pasta `/n8n_workflows`.

6. **Para rodar aplicações Streamlit:**
   ```bash
   streamlit run app.py
   ```
